This simple maze game was developed by Tyson Lindhardt using Javascript, Jquery and Bootstrap. The maze was generated with a Randomized Kruskal's Algorithm and then solved so I can count score. The score is 5 points per correct space, -1 point for moving to the adjacent path and -2 points for moving anywhere else. The maze will accept any number and generate a maze from 4x4 - 100x100. There are several hot keys in the game, you can toggle breadcrumbs with b, shortest path with p, score with y and hint with h.
